可搜索的select选择框
======================
实例为可搜索的select选择框控件,可以实现与服务器交互，动态搜索。
脚本引用的jquery-editable-select插件是稍作修改的。
脚本使用php语言，直接放到lampp中即可访问，供大家学习参考。
